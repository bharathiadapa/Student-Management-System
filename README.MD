# Student Management System

Simple Java console app to manage student records.  
Built for practice and Myntra Hackathon application.

## Features
- Add Student
- Display All Students  
- Search by ID
- Delete by ID

## How to Run
```bash
javac StudentManagementSystem.java
java StudentManagementSystem
